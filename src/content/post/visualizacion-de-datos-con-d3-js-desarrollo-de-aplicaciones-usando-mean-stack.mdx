El pasado 5 de abril fue una  noche éxitosa para PereiraJs, con nuevos speakers invitados y un gran público con sed y ansias de aprender mucho más sobre tecnologías basadas en Javascript, se abordaron temas relacionados con:

**1. Visualización de datos en la web con d3.js by [Juan Sebastián Vega](https://twitter.com/sebasvega95)**

Los datos pueden contener mucha información relevante para notar patrones o tomar decisiones. Para poder analizar los datos y contar una historia con ellos, muchas veces es mejor visualizarlos de alguna manera que simplemente mirar números y estadísticas. Existen infinidad de herramientas para generar y compartir estas visualizaciones de datos, pero muchas son costosas y requieren que quienes deseen ver nuestras visualizaciones también tengan acceso a estas herramientas.

Hoy en día la web es el medio digital que es accesible al mayor número de personas. Tiene sentido crear y distribuir visualizaciones de datos en la web, para que cualquier usuario pueda ver la historia que deseamos narrar con datos. D3.js de lejos la librería más utilizada para crear visualizaciones de datos interactivas y dinámicas en la web. Nos permite unir parte de nuestra página con los datos, para así aplicar transformaciones basadas en los datos. En esta charla aprenderás los principios detrás de D3.js y cómo utilizarla para crear tus propias visualizaciones en la web.

Slides disponibles en: https://bit.ly/2ExNRL7.

**2. Desarrollo de aplicaciones usando MEAN Stack by [Yeferson Gaitan](https://twitter.com/YefriGaitan)**

Uno de los stacks mas comunes en la actualidad es el MEAN stack. MEAN es un acrónimo para MongoDB, ExpressJS, Angular y NodeJS, y es un concepto que ha ganado gran fuerza gracias al crecimiento y popularidad de cada una de estas tecnologías, ademas, el hecho de que solo necesitas conocer un lenguaje de programación (JavaScript) para empezar a desarrollar tu aplicación web.

En esta charla hablaremos sobre las funciones principales de cada una de estas tecnologías, y veremos como vincularlas a nuestro proyecto, de tal manera que nos permita desarrollar de una forma amigable y divertida. Contaremos algunas experiencias y mostraremos ejemplos de aplicaciones muy útiles que fueron desarrolladas usando este stack.

Slides disponibles [en este enlace](https://docs.google.com/presentation/d/1Lt_1hVTBpuylyefppGKTr0ahZqK5LGkR7kVTL5J8k5A/edit#slide=id.g3637f5fcf8_0_177). 
Código disponible [en este enlace](https://github.com/Yefri97/MEANStack-PereiraJS).

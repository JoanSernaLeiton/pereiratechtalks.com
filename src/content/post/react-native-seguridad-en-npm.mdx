Nuevamente nuestros speakers nos descrestaron en una gran noche de mucho Javascript, compartiendo conocimientos sobre:

1. **Introduccion a React Native by [Carlos Álvaro González](https://twitter.com/caal_15)**

Hoy en día el tráfico en la red a través de dispositivo móviles supera en gran medida al de las computadoras personales, razón por la cual se ha vuelto tendencía la filosofía "Mobile First, donde se diseña pensando para móviles primero.

Por lo anterior, en esta charla hablaremos sobre React Native, un framework de javascript que permite crear aplicaciones móviles híbridas basadas en React y que compilan a código nativo en iOS y Android , mejorando el rendimiento enormemente comparado con otros frameworks para crear apps híbridas, como los basados en Phonegap.

Más información en: https://facebook.github.io/react-native/

![react-native-1](~/assets/images/posts/content/react-native-1.jpg)

2. **Exponiendo credenciales de npm sin ningún fallo by [Daniel Aristizabal](https://twitter.com/cronopio2)**

Malas prácticas como exponer variables de entorno con credenciales de autenticación en repositorios de npm, suelen ser un error común que comenten algunos desarrolladores que pueden traer graves consecuencias.

En esta charla aprenderemos sobre buenas prácticas de seguridad y como obtener contraseñas débiles o expuestas en npm.

Más información en: https://github.com/ChALkeR/notes/blob/master/Gathering-weak-npm-credentials.md
